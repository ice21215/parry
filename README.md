# Parry Skills Repository

這是一個專門存放自動化與資料處理「技能 (Skills)」的程式碼庫。

目前包含的技能模組：

## 1. 單項金額總支出統計 (Item Total Expense Calculator)

這是一個專為分析 Excel 報表（如「零用金收支明細表」）所設計的小工具。它能自動讀取表格內的資料，針對相同項目的花費進行彙整，並依照金額由大到小排序，讓各項目的總支出一目了然！

### ✨ 核心功能
*   **智慧欄位辨識**：自動尋找表格中的「項目/說明」以及「支出/金額」等相關欄位，無需固定格式。
*   **日期追蹤**：會同步抓取花費發生的日期，方便回溯與查帳。
*   **精美終端機輸出**：透過色彩標示、對齊排版，直接在終端機或命令提示字元 (cmd) 呈現清楚易讀的統計報告。

### 🚀 如何使用
1.  **安裝依賴套件**：
    確保環境已安裝 NodeJS，然後安裝試算表解析套件：
    ```bash
    npm install xlsx
    ```

2.  **執行腳本**：
    透過 `node` 執行該腳本並帶入 Excel 檔案的絕對路徑：
    ```bash
    node skills/item-total-expense/scripts/calculate_item_expense.js "C:\路徑\到\您的\記帳表.xlsx"
    ```
    
    *(小撇步：Windows 用戶可以寫一個 `.bat` 批次檔將此腳本包裝起來，未來只需把 Excel 檔案「拖曳」到 `.bat` 檔案上就能瞬間完成計算！)*
