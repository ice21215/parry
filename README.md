# Aegis 智慧法規檢索分析系統

Aegis 是一個具備科技感、直覺且高效的「法規檢索與解析」靜態網頁應用。它利用預先爬取的法規資料庫（目前主要為《職業安全衛生設施規則》），幫助使用者快速搜尋、解析法條，並附帶重點白話說明，讓法規知識不再生硬難懂。

## ✨ 核心特色

- **UI UX PRO MAX EXPERT+**：採用深色模式 (Dark Mode)、玻璃擬物化 (Glassmorphism) 設計，並搭配流暢微動畫，提供百萬級系統的操作體驗。
- **精準條號檢索**：支援特定格式的搜尋，如輸入 `《職業安全衛生設施規則》 第 226 條`，系統會直接將該條文明確呈現，剃除無關資訊。
- **複合關鍵字搜尋**：可輸入如 `合梯 堅固` 等多個關鍵字，系統會透過交集比對為您過濾最符合的法規。
- **白話解析與重點提示**：除了法規原文，更額外標註重點解析，協助非法律背景使用者快速掌握合規要求。
- **純靜態架構 (Static Website)**：免後端伺服器！直接點開 `index.html` 即可運行，且非常適合部署至 GitHub Pages。

## 📂 專案結構

```
regulation-search/
│
├── index.html       # 系統首頁 (負責 UI 框架與結構)
├── styles.css       # 樣式表 (深色科技主題、玻璃特效、動畫)
├── app.js           # 核心邏輯 (負責字串比對、搜尋演算、DOM 動態渲染)
└── db.js            # 法規資料庫 (由爬蟲自動生成的 JSON 陣列)
```

## 🚀 快速開始

### 本地端運行
本系統不需要安裝任何相依套件 (Dependencies) 或啟動本地伺服器，只要：
1. 將專案 Clone 或下載至本地。
2. 連續點擊兩下 `index.html` 檔案。
3. 開始檢索法規！

### 部署至 GitHub Pages
因系統為純靜態檔案，可以直接在 GitHub 上開啟 GitHub Pages 服務：
1. 進入此 GitHub Repository 的 **Settings**。
2. 點擊左側選單的 **Pages**。
3. 在 **Build and deployment** > Source 中選擇 `Deploy from a branch`。
4. Branch 選擇 `main` 與 `/ (root)`，然後按下 **Save**。
5. 等待約一分鐘後，您即可獲得專屬的線上法規查詢網址！

## 🔍 搜尋範例試驗

您可以直接在搜尋框輸入以下範例：
1. `合梯`（測試單一關鍵字，會帶出所有提到合梯的規範）
2. `《職業安全衛生設施規則》 第 227 條`（測試精準法條查詢）
3. `護欄 堅固`（測試多關鍵字複合過濾）

## 🛠️ 開發與維護指南 (擴充法規)

目前的法規資料是由後端爬蟲腳本動態從「全國法規資料庫」爬取並生成。若您未來需要替換或擴增其他法規（如《勞動基準法》），只需遵循以下 JSON 格式覆寫或擴充 `db.js` 中的 `regulationsDB` 陣列即可：

```javascript
[
    {
        "keywords": ["關鍵字1", "關鍵字2"],
        "lawName": "法規名稱",
        "article": "第 X 條",
        "originalText": "法規原文內容...",
        "summary": "法規重點白話解析..."
    }
]
```

## ⚠️ 免責聲明

此系統的法規資訊與白話摘要為輔助查詢性質，可能無法即時反映立法機關最新之修法變動。應用程式內之分析結果僅供參考，不構成正式法律意見，若是涉及實體權益，仍請以主管機關或專業律師見解為準。
